# Automated Q2C Order Validation Analytics System

## Project Overview

This project automates the Quote to Cash order validation process using Excel VBA and Power BI. It helps identify missing order details, calculate SLA delays, update order status, and monitor operational bottlenecks through an interactive dashboard.

## Problem Statement

Manual order validation takes time and increases the chance of errors. Teams may miss incomplete orders, SLA breaches, and delayed processing. This project solves that problem by automating validation and creating clear visibility into order performance.
